# GEMINI.md - Gemini向けプロジェクトガイド

このドキュメントは、Geminiエージェントがこのプロジェクトで作業するためのガイドラインを提供します。

## 全体的な開発ガイドライン

このプロジェクトでの作業は、プロジェクト固有の指示に加え、[AGENT_GUIDELINE.md](AGENT_GUIDELINE.md) に記載されている普遍的な開発ガイドラインに従ってください。

## プロジェクト概要

Node.jsと素のJavaScriptで構築された、シンプルな家計簿アプリケーションです。ユーザーは収入と支出を記録し、閲覧することができます。

## 技術スタック

- **バックエンド:** Node.js, better-sqlite3
- **フロントエンド:** Vanilla JavaScript (ES Modules), Chart.js
- **パッケージマネージャー:** npm

## 開発

### 前提条件

- Node.js
- npm

### セットアップ

依存関係をインストールするには、次のコマンドを実行します:
```bash
npm install
```

### アプリケーションの実行

開発サーバーを起動するには、次のコマンドを実行します:
```bash
node server.js
```
アプリケーションは `http://localhost:3000` で利用可能になります。

## テスト

テストを実行するには、npmスクリプトを使用します:
```bash
npm test
```

## コーディング規約

- バックエンドとフロントエンドの両方でESモジュール (`import`/`export`) を使用します。
- 既存のコードスタイルとフォーマットに従ってください。
- 新しい関数やモジュールには、その目的、パラメータ、戻り値を説明するJSDocコメントを追加してください。

## 主要なファイル

- `server.js`: バックエンドサーバーのメインエントリーポイント。APIルートと静的ファイルの配信を処理します。
- `db.js`: SQLiteデータベースの接続とクエリを管理します。
- `frontend/`: すべてのフロントエンドファイルが含まれています。
  - `frontend/index.html`: メインのHTMLファイル。
  - `frontend/App.js`: フロントエンドの主要なアプリケーションロジック。ルーティングを処理します。
  - `frontend/components/`: フロントエンドコンポーネント（例: Dashboard, Forms）が含まれています。
